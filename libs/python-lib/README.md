# Activate API Models for Python

This package contains auto-generated DTO models and API clients for Python applications.

## Installation

```bash
pip install activate_api_models-1.0.0-py3-none-any.whl
```

## Usage

```python
from activate_api_models.models import DemoMessage, Item
from activate_api_models.apis import DefaultApi

# Use the models and API
```

## Build

To build the wheel package:

```bash
python -m build
```

The wheel file will be created in the `dist/` directory.
